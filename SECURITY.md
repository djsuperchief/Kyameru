# Security Policy

## Supported Versions

The Kyameru policy is to support the latest version of the product.

| Version | Supported          |
| ------- | ------------------ |
| 0.0.18-RC  | :white_check_mark: |
| < 0.0.18-RC  | :x: |

## Reporting a Vulnerability

To raise a vulnerability, please raise a ticket and tag it appropriatly.
